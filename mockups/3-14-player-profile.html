S006 decisions · S007 refinements (cross-ref design spec §3.14)

  • P1 · "Best season finish" → COMPUTE LIVE. Client-side scan of v_season_standings across all seasons on profile load. Materialize into a new view in Phase 2 only if latency becomes painful.
  • P2 · Recent-matches row tap → FULLY TAPPABLE. Normal tappable affordance in Phase 1. Destination resolved as P5 (match-detail bottom sheet).
  • P3 · Email visibility → EDIT SHEET ONLY. Hero stays clean; email surfaces only inside the edit sheet.
  • P4 · "You" indicator → DROPPED. Self-view is implied by the edit pencil + entering via the bottom-nav profile tab. Removed from the hero.
  • P5 · Recent-matches row tap destination → READ-ONLY MATCH-DETAIL SHEET (§3.15). Full Depth-B spec + mockup for the sheet deferred past S007; stub carried in the design spec and Phase 1 backlog.
  • W-D-L colour triplet (S006 review). All W-D-L renderings use green W / grey D / red L (same palette as §3.2 last-5 strip and result badges). Applies to season-stats and anywhere the triplet appears.
  • Copy tweak (S006 review). Last-5 strip card label reads "Last 5 games (season)" for clarity (previously "Last 5 (season)").
  • S007 · W-D-L alignment fix. Loss digit previously inherited .kpi .l label styling (9px + margin-top). Specificity-override added so W/D/L all render at 17px with matching baseline.
  • S007 · Last-5 circle centering fix. Letter-spacing reset to 0 + explicit line-height: 1 on circles — glyph now sits centred in the 24 px disc.
  • S007 · MP stat added to Season stats. Matches-played joins the 6-KPI grid (Points · MP · W-D-L · Goals · MOTM · Late-cancel). Rank removed from grid — already shown as the rank 1st 🥇 hint at the card header.
  • S007 · Totals card removed; Achievements card added. Totals duplicated Season stats without context. Achievements surfaces career highlights instead: MOTMs · best W-streak · total goals · yellows · reds · longest L-streak. Scope dropdown retired with the Totals card.
