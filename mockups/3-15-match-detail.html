§3.15 Depth-B · S009 (cross-ref design spec §3.15)

  • Upgrade path. S006 shipped the STUB (scope + layout list only). S008 locked the open questions: W/D/L chip = profile-owner perspective; guest rows lighter + inline stats only (no rating chip, no description); wide-viewport = 640 × 80 vh provisional.
  • Entry points. §3.14 Recent matches row tap (primary, carries profile_id) · §3.7 Poll "last match" tile · notification deep links (match_approved, motm_awarded) · §3.13 Leaderboard drill-down reserved for Phase 2 (no profile context → chip omitted).
  • Read-only. No edits on this sheet in Phase 1. Admin edits live on the Admin → Matches dashboard (spec not yet written).
  • W/D/L chip perspective rule. Chip reflects how the profile-owner ended up in this match. Late-cancel → chip omitted (penalty strip carries the context). No profile context → chip omitted entirely; header shows only the scoreline.
  • Guest rendering contract. Italic name, gold avatar, "+1 · invited by …" subtitle, inline goals/cards/MOTM star (same as members). NO S007 rating chip here · NO description here (those stay poll-screen recruitment context per S008 decision). Guests CAN be MOTM per §2.3 — MOTM row renders the guest italic + gold.
  • Captain marker. Small (C) pill inline before the name. Captains render as the first row in their team section.
  • Late-cancel strip. Renders only if any match_players.late_cancel_penalty = true exists for the match. Shows 🕒 late cancel · name · −1 pt. 7-day automatic ban enforcement deferred to Phase 2 and NOT surfaced on this read-only sheet.
  • Durable W-D-L colour. Green W / grey D / red L — same palette as §3.2 last-5, §3.13 leaderboard, §3.14 profile. Chips on this sheet match.
  • Dates. Footer renders DD/MMM/YYYY uppercase per durable preference; storage stays ISO.
  • Safe-area. Bottom padding of sheet = calc(12px + var(--safe-bottom)) so the footer clears the iOS home indicator. Mobile sheet never reaches the top edge (max-height 90%); wide-viewport modal sits comfortably inside the viewport.
  • Phase-2 deferred. Share-as-PNG · goal timeline · assists · comments · compare-with-me · admin shortcuts.
